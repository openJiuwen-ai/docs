## common

`openjiuwen.agent.common` is the module in openJiuwen used to define shared foundational components. It provides the core enum constants and data structure (schema) definitions that are central to Agents.

**Classes**:

| CLASS                                                                                       | DESCRIPTION                                                             |
| ------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| [ControllerType](./common/enum.md#class-openjiuwenagentcommonenumcontrollertype)            | Defines the types of controllers.                                       |
| [TaskType](./common/enum.md##class-openjiuwenagentcommonenumtasktype)                     | Defines the types of subtasks.                                          |
| [TaskStatus](./common/enum.md#class-openjiuwenagentcommonenumtaskstatus)                    | Defines the execution states of a task.                                 |
| [ReActControllerStatus](./common/enum.md#class-openjiuwenagentcommonenumreactcontrollerstatus) | Defines the states of the ReAct controller.                             |
| [WorkflowSchema](./common/schema.md#class-openjiuwenagentcommonschemaworkflowschema)        | Defines the metadata and configuration of a workflow.                   |
| [PluginSchema](./common/schema.md#class-openjiuwenagentcommonschemapluginschema)            | Defines the metadata and configuration of a plugin.                     |
| [McpSchema](./common/schema.md#class-openjiuwenagentcommonschemamcpschema)                  | Defines the metadata and configuration of MCP (Model Context Protocol). |
