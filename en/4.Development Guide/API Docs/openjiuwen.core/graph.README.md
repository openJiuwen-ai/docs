# graph

`openjiuwen.core.graph` is the workflow execution graph of openJiuwen.

**Classes**：

| CLASS | DESCRIPTION |
|------|-------------|
| [Graph](./graph/base.md) | Base class used to construct workflow execution graphs. |

**Alias**：

| ALIAS | DESCRIPTION |
|------|-------------|
| [Router](./graph/base.md) | A router for conditional edges in a graph or workflow, used to determine the target node or list of target nodes for a conditional edge. |
