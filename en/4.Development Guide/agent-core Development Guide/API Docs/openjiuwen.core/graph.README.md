# graph

`openjiuwen.core.graph` is the workflow execution graph of openJiuwen.

**Classes**：

| CLASS              | DESCRIPTION            |
|--------------------|------------------------|
| [Graph](./graph/graph.md) | Graph class is the base class used to construct workflow execution graphs. |

**Alias**：

| ALIAS                     | DESCRIPTION                                                          |
|---------------------------|----------------------------------------------------------------------|
| [Router](./graph/graph.md) | Router is the routing for conditional edges in a graph or workflow, used to determine the target node or list of target nodes for a conditional edge. |
