# openjiuwen.core.controller

## class openjiuwen.core.controller.schema.FileDataFrame

File data frame, used for transmitting files in events and task input/output.

- **type** (Literal["file"]): Data frame type, fixed as `file`.
- **name** (str): File name.
- **mimeType** (str): MIME type, e.g., `image/png`, `application/pdf`.
- **bytes** (Optional[bytes]): File binary content; mutually exclusive with `uri`, choose one.
- **uri** (Optional[str]): File URI (local path or remote address); mutually exclusive with `bytes`.

## class openjiuwen.core.controller.schema.dataframe.JsonDataFrame

Represents JSON format data frame, used for passing structured data (e.g., configuration, API responses) in events and task input/output.

- **type** (Literal["json"]): Data frame type, fixed as `json`.
- **data** (Dict[str, Any]): JSON dictionary content.

## class openjiuwen.core.controller.schema.dataframe.TextDataFrame

Represents text type data frame, used for passing plain text content (e.g., user input, prompt fragments, log fragments) in streaming or batch scenarios of events and task input/output.

- **type** (Literal["text"]): Data frame type, fixed as `text`.
- **text** (str): Text content.

## class openjiuwen.core.controller.schema.DataFrame

Data frame union type, covering `TextDataFrame`, `FileDataFrame`, `JsonDataFrame`.

## class openjiuwen.core.controller.schema.Event

Event base class, describing common events handled by controllers.

- **metadata** (Optional[Dict[str, Any]]): Event metadata.

## class openjiuwen.core.controller.schema.EventType

Event type enumeration, defining four types of event identifiers handled by controllers.

- **INPUT**: User input event.
- **TASK_INTERACTION**: Interaction event during task execution.
- **TASK_COMPLETION**: Task completion event.
- **TASK_FAILED**: Task failure event.

## class openjiuwen.core.controller.schema.InputEvent

- **event_type** (EventType): Fixed as `EventType.INPUT`.

- **input_event_id** (Optional[str]): Associated input event ID.

## class openjiuwen.core.controller.schema.IntentType

Intent type enumeration, representing different intent categories.

- **CREATE_TASK**: Create a new task (will interrupt any currently executing task first).
- **PAUSE_TASK**: Pause the currently executing task.
- **RESUME_TASK**: Resume a previously paused task (status changes from paused to submitted).
- **CONTINUE_TASK**: Continue execution based on a completed task, new task depends on the completed task's context.
- **SUPPLEMENT_TASK**: Provide supplementary information for tasks requiring user input, then continue execution.
- **CANCEL_TASK**: Cancel the currently executing task.
- **MODIFY_TASK**: Modify parameters/configuration of a task in execution, resubmit after modification.
- **SWITCH_TASK**: Interrupt current task and execute a new task.
- **UNKNOWN_TASK**: Unrecognized intent, requires user clarification.

## class openjiuwen.core.controller.schema.Intent

Single intent object, describing "what the user wants to do", and containing context information needed to execute the intent.

- **intent_type** (IntentType): Intent type.
- **event** (Event): Associated input event.
- **target_task_id** (Optional[str]): Target task ID.
- **target_task_description** (Optional[str]): Task description (required for CREATE_TASK/SWITCH_TASK).
- **depend_task_id** (List[str]): Dependent task ID list (required for CONTINUE_TASK).
- **supplementary_info** (Optional[str]): Supplementary information (required for SUPPLEMENT_TASK).
- **modification_details** (Optional[str]): Modification content (required for MODIFY_TASK).
- **confidence** (float): Confidence level, must be between 0.0–1.0, default value 1.0.
- **metadata** (Optional[Dict[str, Any]]): Additional metadata, default empty dictionary.
- **clarification_prompt** (Optional[str]): Clarification prompt (required for UNKNOWN_TASK).

## class openjiuwen.core.controller.schema.TaskStatus

Task status enumeration:
- **SUBMITTED**: Submitted, waiting for execution.
- **WORKING**: Executing.
- **INPUT_REQUIRED**: Requires user input/supplementary information.
- **COMPLETED**: Completed.
- **FAILED**: Execution failed.
- **PAUSED**: Paused, can be resumed.
- **CANCELED**: Canceled.
- **WAITING**: Waiting (e.g., dependent tasks not completed).
- **UNKNOWN**: Unknown status.

## class openjiuwen.core.controller.schema.Task

Task data model, describing task identification, status, and input/output.

- **session_id** (str): Session ID the task belongs to, cannot be empty.
- **task_id** (str): Task ID, cannot be empty, and cannot be the same as `parent_task_id`.
- **task_type** (str): Task type, used for selecting executor, cannot be empty.
- **description** (Optional[str]): Task description.
- **priority** (int): Priority, smaller values indicate higher priority. Value range is greater than or equal to zero.
- **inputs** (Optional[List[Event]]): Associated input event list.
- **outputs** (List[ControllerOutputChunk]): Streaming output chunk list.
- **status** (TaskStatus): Task status. If FAILED, must provide non-empty `error_message`; if INPUT_REQUIRED, must provide `input_required_fields`.
- **parent_task_id** (Optional[str]): Parent task ID. Cannot equal `task_id`.
- **context_id** (Optional[str]): Context association ID.
- **input_required_fields** (Optional[Union[Dict[str, Any], BaseModel]]): Field definitions required when status is `input-required`.
- **error_message** (Optional[str]): Error message on failure.
- **metadata** (Optional[Dict[str, Any]]): Additional metadata.
- **extensions** (Optional[Dict[str, Any]]): Extension fields.

## class openjiuwen.core.controller.modules.TaskScheduler

```
class openjiuwen.core.controller.modules.TaskScheduler(config: ControllerConfig,
                                                                      task_manager: TaskManager,
                                                                      context_engine: ContextEngine,
                                                                      ability_manager: AbilityManager,
                                                                      event_queue: EventQueue,
                                                                      card: AgentCard)
```

Task scheduler, responsible for scheduling/executing/pausing/canceling tasks, and producing execution results in streaming mode.

**Parameters**:

* **config** (ControllerConfig): Controller configuration. Default value: none (must be explicitly passed).
* **task_manager** (TaskManager): Task manager instance.
* **context_engine** (ContextEngine): Context engine.
* **ability_manager** (AbilityManager): Ability manager (tools/workflows, etc.).
* **event_queue** (EventQueue): Event queue, used for publishing task-related events.
* **card** (AgentCard): Agent card information, used for event publishing identification.

### async start() -> None

Start background scheduling loop `schedule`, begin periodic scanning of `SUBMITTED` tasks.

### async stop() -> None

Stop scheduler: cancel unfinished tasks, terminate scheduling loop and wait for cleanup.

### async schedule() -> None

Scheduling main loop: pull `SUBMITTED` tasks at `schedule_interval`; create async execution wrapper `_execute_task_wrapper` for each task within `max_concurrent_tasks` limit; brief sleep when no tasks; wait for all running tasks to complete when stopping.

### async execute_task(task_id: str, session: Session) -> None

Execute single task main flow: get task → create executor → set status `WORKING` → iterate `execute_ability` streaming output → update status and publish events based on `payload.type`.

**Parameters**:

* **task_id** (str): Task ID.
* **session** (Session): Session object.

**Exceptions**:

* `AGENT_CONTROLLER_TASK_EXECUTION_ERROR`: Task does not exist or executor is missing.

### async pause_task(task_id: str) -> bool

If task is running, call executor `can_pause`/`pause` and cancel corresponding `asyncio.Task`, update status to `PAUSED`.

**Parameters**:

* **task_id** (str): Task ID.

**Returns**:

**bool**, whether pause was successful.

### async cancel_task(task_id: str) -> bool

Similar to `pause_task`, call executor `can_cancel`/`cancel` then cancel coroutine and set status to `CANCELED`.

**Parameters**:

* **task_id** (str): Task ID.

**Returns**:

**bool**, whether cancellation was successful.

## class openjiuwen.core.controller.ControllerConfig


Controller configuration, covering scheduling, task, event queue, and intent recognition parameters.

- **max_concurrent_tasks** (int): Maximum concurrent tasks, default value is 5, 0 means unlimited.
- **schedule_interval** (float): Scheduling scan interval (seconds), default value is 1.0.
- **task_timeout** (Optional[float]): Task timeout (seconds, minimum 600), default value is None.
- **default_task_priority** (int): Default task priority, default value is 1.
- **enable_task_persistence** (bool): Whether to enable task state persistence, default value is False.
- **event_queue_size** (Optional[int]): Event queue size, default value is 10000.
- **event_timeout** (Optional[float]): Event processing timeout (seconds), default value is 300.
- **enable_intent_recognition** (bool): Whether to enable intent recognition, default value is False.
- **intent_llm_id** (str): Intent recognition LLM ID.
- **intent_confidence_threshold** (float): Intent confidence threshold, default value is 0.7.
- **intent_type_list** (List[str]): List of supported intent types.
