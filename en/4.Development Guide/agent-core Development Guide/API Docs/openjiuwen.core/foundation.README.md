# foundation

`openjiuwen.core.foundation` provides the foundational capabilities module of openJiuwen, including core functions such as large model invocation, tools, prompt templates, and storage.

**Modules**:

| MODULE | DESCRIPTION | Detailed API |
|--------|-------------|--------------|
| [llm](./foundation/llm.README.md) | Large model invocation and management module. | [llm.md](./foundation/llm/llm.md) |
| [tool](./foundation/tool.README.md) | Tool definition and invocation module. | [tool.md](./foundation/tool/tool.md) |
| [prompt](./foundation/prompt.README.md) | Prompt template module. | [template.md](./foundation/prompt/template.md) |
| [store](./foundation/store.README.md) | Storage abstraction module. | [store.md](./foundation/store/store.md) |
