# stream

`openjiuwen.core.stream` is the module that enables streaming output functionality in openJiuwen. It significantly improves the real-time performance and efficiency of data processing. The openJiuwen framework provides multiple built-in streaming modes for direct use.

**Classes**：

| CLASS | DESCRIPTION |
|------|-------------|
| [BaseStreamMode](./stream/base.md) | Enumeration class defining streaming output modes in openJiuwen. |
| [OutputSchema](./stream/base.md) | Data class defining the standard streaming output data format in openJiuwen. |
| [CustomSchema](./stream/base.md) | Data class for user-defined streaming output data formats. |
| [TraceSchema](./stream/base.md) | Data class for streaming output data formats used for debugging and tracing. |
