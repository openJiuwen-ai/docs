# API Documentation

This section organizes API documentation according to the source code package structure, ensuring a one-to-one correspondence with `agent-core/openjiuwen`.

| Package/Module | Description |
|---|---|
| [`openjiuwen.core`](./openjiuwen.core.README.md) | Core capabilities: workflow, session, runner, retrieval, memory, etc. |
| [`openjiuwen.dev_tools`](./openjiuwen.dev_tools.README.md) | Development tools: prompt_builder, tune (prompt generation/optimization and tuning). |
| [`openjiuwen.extensions`](./openjiuwen.extensions.README.md) | Optional extensions: such as message_queue. |
