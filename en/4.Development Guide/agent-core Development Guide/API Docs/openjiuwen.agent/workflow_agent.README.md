### `workflow_agent`

`openjiuwen.agent.workflow_agent` is an Agent module in OpenJiuwen that provides capabilities for creating and executing a `WorkflowAgent`.

**Classes**:

| CLASS                                                                                 | DESCRIPTION                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| ------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [WorkflowAgent](./workflow_agent.md#class-openjiuwenagentworkflow_agentworkflowagent) | `WorkflowAgent` is an agent specialized in multi-step, task-oriented workflow automation. It efficiently executes complex tasks by strictly following user-defined workflows. Users can predefine clear task steps, execution conditions, and role assignments, decomposing tasks into multiple executable subtasks or tools. Through topological connections and data passing between components, the agent incrementally advances the entire workflow and ultimately produces the expected results. It emphasizes standardized and efficient execution based on predefined processes, making it suitable for scenarios where tasks are well-structured and can be broken down into multiple steps. |
