The document introduces the installation methods for openJiuwen.

  ![image](./images/openJiuwen-install-flow.png)
<center>Figure: openJiuwen installation workflow</center>  

To help you get started quickly, the community provides the following two installation options: 
* ​[Install via Docker](./Install%20via%20Docker/README.md)​：Recommended for users who want rapid deployment and a quick hands-on experience. The community has packaged all dependencies into containers.
* ​[Local Installation](./Local%20Installation/README.md)​：Recommended for developers, contributors, or users who need code customization. This method requires manual dependency setup and allows full control over the development environment for debugging and modification.
