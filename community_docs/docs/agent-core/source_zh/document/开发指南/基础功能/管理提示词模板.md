`TemplateManager`模块实现提示词模板的全生命周期管理，包括提示词模板的注册、更新、查询、删除、批量创建等能力。主要包括以下几个功能：

- 提示词模板注册/更新：提供将提示词模板注册到内存中以及更新提示词模板的能力。
- 提示词模板查询/删除：提供查询和删除提示词模板的能力。
- 提示词模板批量创建：支持用户将指定目录下，符合提示词模板格式的模板文件，批量注册至内存中。

# 注册/更新提示词模板

实例化Template模板类后，可以使用`TemplateManager`的`register`方法将该提示词模板注册到内存中，其中`register`方法提供的`force`参数表示是否强制更新提示词模板，默认是False。当提示词模板尚未被注册，force参数为`True`或`False`都表示将提示词模板注册到内存中，当提示词模板已被注册，force参数为`True`则表示强制更新提示词模板，force参数为`False`则表示不允许更新，会抛`JiuWenBaseException`异常。示例代码如下：

首次注册提示词模板：

```python
from jiuwen.core.utils.prompt.index.template_store.template_store import Template
from jiuwen.core.utils.prompt.template.template_manager import TemplateManager

# 实例化Template提示词模板类
template = Template(name="test_template_register", content=[{"role": "system", "content": "testing for template register."}], filters={"model_name": "pangu"})

# 实例化TemplateManager提示词模板管理类
template_manager = TemplateManager()

# 此时不存在提示词模板，表示首次注册
template_manager.register(template=template)
print("首次注册：", template_manager.get(name="test_template_register", filters={"model_name": "pangu"}))
```

输出结果如下：

```python
首次注册：Template(name='test_template_register', content=[{'role': 'system', 'content': 'testing for template register.'}], filters={'model_name': 'pangu'})
```

提示词已存在时，更新提示词模板：

```python
# 此时存在提示词模板，force参数为True，表示更新提示词模板
template.content = "testing for template update."
template_manager.register(template=template, force=True)
print("更新：", template_manager.get(name="test_template_register", filters={"model_name": "pangu"}))
```

输出结果如下：

```
更新：Template(name='test_template_register', content=[{'role': 'system', 'content': 'testing for template update.'}], filters={'model_name': 'pangu'})
```

重复注册已存在的提示词模板：

```python
# 此时存在提示词模板，force参数为False，表示不允许更新提示词模板
try:
    template_manager.register(template=template, force=False)
except JiuWenBaseException as error:
    print("重复注册：", error)
```

输出结果如下：

```
重复注册： [102101] Template: test_template_register is duplicated to register
```

# 查询提示词模板

使用`TemplateManager`的`get`方法可以获取指定`name`的提示词模板，示例代码如下：

```python
from jiuwen.core.utils.prompt.index.template_store.template_store import Template
from jiuwen.core.utils.prompt.template.template_manager import TemplateManager

template_manager = TemplateManager()
template = Template(name="test_template_register", content=[{"role": "system", "content": "testing for template register."}])
# 注册name为test_template_register的提示词模板
template_manager.register(template=template, force=True)
# 注册后获取name为test_template_register的提示词模板
print(template_manager.get(name="test_template_register"))
```

输出结果：

```python
Template(name="test_template_register", content=[{"role": "system", "content": "testing for template register."}])
```

如果不存在指定`name`的提示词模板，则会抛`JiuWenBaseException`异常，示例代码如下：

```python
from jiuwen.core.utils.prompt.template.template_manager import TemplateManager
from jiuwen.core.common.exception.exception import JiuWenBaseException

template_manager = TemplateManager()
try:
    # 未注册name为test_template_not_exist的提示词模板，但尝试获取
    print(template_manager.get(name="test_template_not_exist"))
except JiuWenBaseException as e:
    print(f"异常信息： {e}")
```

输出结果：

```python
异常信息： [102102] Template not found
```

# 删除提示词模板

使用`TemplateManager`的`delete`方法可以删除指定`name`的提示词模板，如果指定删除的`name`尚未被注册，则会抛`JiuWenBaseException`异常。示例代码如下：

```python
from jiuwen.core.utils.prompt.index.template_store.template_store import Template
from jiuwen.core.utils.prompt.template.template_manager import TemplateManager
from jiuwen.core.common.exception.exception import JiuWenBaseException

template_manager = TemplateManager()
template = Template(name="test_template_delete", content=[{"role": "system", "content": "testing for template delete."}])
# 注册name为test_template_delete的提示词模板
template_manager.register(template=template, force=True)

# 注册后查询name为test_template_delete的提示词模板可正常获取
print(template_manager.get(name="test_template_delete"))

# 删除提示词模板
template_manager.delete(name="test_template_delete")
try:
    # 删除后再次查询name为test_template_delete的提示词模板
    print(template_manager.get(name="test_template_delete"))
except JiuWenBaseException as e:
    print(f"删除后查询异常: {e}")

# 再次删除
try:
    print(template_manager.delete(name="test_template_delete"))
except JiuWenBaseException as e:
   print(f"删除不存在的提示词模板异常: {e}")
```

输出结果：

```python
Template(name="test_template_register", content=[{"role": "system", "content": "testing for template delete."}])
删除后查询异常: [102102] Template not found 
删除不存在的提示词模板异常: [102102] Template: test_template_delete not found to delete
```

# 批量注册提示词模板

使用`TemplateManager`的`register_in_bulk`接口可以将`dir_path`路径下的提示词模板批量注册至内存中，每个提示词模板文件的注册名称为其文件名，后续可通过该文件名查询已注册的提示词模板信息。用户需提前在路径下准备提示词模板文件，以`.pr`为后缀，文件内容为提示词模板内容，如提示词目标包含占位符，需要使用双花括号`{{...}}`占位符格式表示。

指定目录`my_template_dir`下存在如下两个文件, `llm_custmize.pr`和`qa_custmize.pr`。`llm_custmize.pr`内容如下：

```text
你是一个智能助手，你为用户生成综合质量（包括有用性，事实性，无害性）极好的回复。

问题是 : {{query}}
```

`qa_custmize.pr`内容如下：

```text
你是一个智能对话助手，负责帮助收集用户信息并分析工具执行结果。你需要生成自然、简洁的对话回复。
## 用户对话历史
{{dialogue_history}}
问题是 : {{query}}
```

使用`register_in_bulk`接口将`my_template_dir`目录下所有提示词模板文件进行注册，并可根据文件名进行查询，示例代码如下：

```python
from jiuwen.core.utils.prompt.template.template_manager import TemplateManager

# dir_path为自定义提示词模板文件路径，存放llm_custmize.pr 和 qa_custmize.pr文件
dir_path = "./my_template_dir"
template_manager = TemplateManager()
# 批量注册提示词模板
template_manager.register_in_bulk(dir_path=dir_path)
# 根据提示词模板文件名查询批量注册的提示词模板信息
print(template_manager.get(name="llm_custmize"))
print(template_manager.get(name="qa_custmize"))
```

输出如下：

```python
Template(name="llm_custmize", content=[{'role': 'system', 'content': '你是一个智能助手，你为用户生成综合质量（包括有用性，事实性，无害性）极好的回复。\n\n问题是 : {{query}}'}])
Template(name="qa_custmize", content=[{'role': 'system', 'content': '`你是一个智能对话助手，负责帮助收集用户信息并分析工具执行结果。你需要生成自然、简洁的对话回复。\n## 用户对话历史\n{{dialogue_history}}\n问题是 : {{query}}'}])
```
