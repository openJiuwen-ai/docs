# OpenJiuWen Studio 0.1.1 到 0.1.2 升级指导

## 升级说明

⚠️ **重要提示：** 当前版本升级不支持数据库中的数据跨版本兼容，请在升级前提前备份所有重要数据。

请在升级前仔细阅读本指导文档，按照步骤完成升级操作。

## 一、本地安装升级

### 1. 切换到新版本分支

1. 进入项目根目录
2. 拉取最新代码：
   ```bash
   git pull
   ```
3. 切换到 0.1.2 版本分支：
   ```bash
   git checkout v0.1.2
   ```

### 2. 前端升级

1. 进入前端项目目录：
   ```bash
   cd frontend
   ```
2. 重新执行依赖安装命令：
   ```bash
   npm install
   ```
3. 启动开发服务器以应用更新：
   ```bash
   npm run dev
   ```

### 3. 后端升级

1. 进入后端项目目录：
   ```bash
   cd backend
   ```
2. 重新执行依赖同步命令：
   ```bash
   uv sync
   ```
3. 激活虚拟环境并启动后端服务：
   ```bash
   source .venv/bin/activate
   python main.py
   ```

## 二、Docker安装升级

### 1. 重新部署容器

#### 1.1 下载0.1.2版本包

根据您的系统架构选择对应版本包：

- x86_64架构下载链接：
  ```
  https://openjiuwen-ci.obs.cn-north-4.myhuaweicloud.com/agentstudio/deployTool_0.1.2_amd64.zip
  ```

- arm64架构下载链接：
  ```
  https://openjiuwen-ci.obs.cn-north-4.myhuaweicloud.com/agentstudio/deployTool_0.1.2_arm64.zip
  ```

#### 1.2 启动OpenJiuWen服务

1. 将下载的版本包解压到安装目录
2. 进入解压后的目录
3. 执行以下命令启动服务：
   ```bash
   ./service.sh up
   ```

如需停止服务，可以使用以下命令：
```bash
./service.sh down
```

> **说明**：详细安装步骤请参考「Docker方式安装」目录下的对应系统安装文档。

## 三、验证升级

完成上述步骤后，请通过以下方式验证升级是否成功：

1. 前端页面能够正常访问
2. 后端服务能够正常响应API请求

如果在升级过程中遇到任何问题，请查看项目日志、安装指导的FAQ或联系项目管理人员。
