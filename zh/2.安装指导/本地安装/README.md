# 本地高级安装

本地高级安装方式提供多种安装方法，适合需要修改代码、进行二次开发或需要灵活配置的开发者。

## 安装方式

本地高级安装提供三种方法：

* **方法一：前后端脚本安装 + 依赖 Docker 启动**：前后端通过脚本安装 Python、Node.js 等环境并源码启动，MySQL、Milvus 等依赖服务通过 Docker 容器启动，适合需要修改代码、进行二次开发的开发者。
* **方法二：全部手动安装**（不推荐）：需要手动安装和配置所有依赖服务，适合需要灵活调整配置的开发者。

各系统的安装文档中已包含这两种方法的详细说明，请根据您的系统选择对应的安装指南：

* [Windows 系统安装](./Windows系统安装.md)
* [MacOS 系统安装](./MacOS系统安装.md)
* [Linux 系统安装](./Linux系统安装.md)

## 前置要求

* 熟悉 Linux/Unix 系统操作
* 了解服务配置和管理
* 具备网络和代理配置经验
* 能够独立解决安装过程中的问题

## 需要安装的依赖服务

* MySQL 数据库
* MinIO 对象存储
* Milvus 向量数据库
* etcd 分布式键值存储
* 其他项目特定的依赖服务

> **注意**：本地高级安装方式门槛较高，需要用户具备较强的技术背景。如无特殊需求，建议优先使用 [Docker 快速安装](../Docker方式安装/README.md)。
