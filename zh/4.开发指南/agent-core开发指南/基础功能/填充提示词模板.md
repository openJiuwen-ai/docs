提示词模板往往包含可填充的占位符，占位符可以指代工具信息、示例、变量等内容。openJiuwen提供了提示词模板类`PromptTemplate`，其中`content`表示模板内容，为必选，使用双花括号`{{...}}`或自定义占位符标记需要后续填充的内容，实际使用中将这些占位符标记的内容替换为具体内容后进行格式化，最终生成完整的提示词prompt。

# 实例化PromptTemplate模板类

PromptTemplate模板类支持以下两种类型`content`参数的输入，分别是`str`（字符串）和`List[BaseMessage]`（BaseMessage列表）。以下介绍两种类型的`content`的实例化方式。

- `str`类型：实例化PromptTemplate模板类，content类型为字符串类型。

  ```python
  from openjiuwen.core.foundation.prompt.template import PromptTemplate
  template = PromptTemplate(content="你是一个精通{{domain}}领域的问答助手\n用户问题：{{query}}")
  ```

- `List[BaseMessage]`类型：实例化PromptTemplate模板类，content类型为BaseMessage列表类型。
  
  ```python
  from openjiuwen.core.foundation.prompt.template import PromptTemplate
  from openjiuwen.core.foundation.llm.schema.message import SystemMessage, UserMessage
  template = PromptTemplate(
      content=[
          SystemMessage(role='system', content='你是一个精通{{domain}}领域的问答助手'),
          UserMessage(role='user', content="用户问题：{{query}}")
      ]
  )
  ```
  
- 自定义占位符（以`str`类型为例）：

  ```python
  from openjiuwen.core.foundation.prompt.template import PromptTemplate
  template2 = PromptTemplate(
     name="custom",
     content="你是一个精通${domain}$领域的小助手！，用户问题：${query}$",
     placeholder_prefix="${",
     placeholder_suffix="}$",
  )
  ```

# 填充PromptTemplate模板

实例化PromptTemplate模板类后，可以调用`.format`方法对`content`进行填充，`{{...}}`或自定义的占位符标记的内容会被实际值正确的替换，从而生成完整的输入内容。示例代码如下：

```python
messages = template.format({"domain": "数学", "query": "1+1=?"}).to_messages()
```
- 通过`str`类型生成的提示词模板，填充后结果为：

  ```python
  [
      UserMessage(role='user', content='你是一个精通数学领域的问答助手\n用户问题：1+1=?', name=None)
  ]
  ```

- 通过`List[BaseMessage]`类型生成的提示词模板，填充后结果为：

  ```python
  [
      SystemMessage(role='system', content='你是一个精通数学领域的问答助手', name=None),
      UserMessage(role='user', content='用户问题：1+1=?', name=None)
  ]
  ```

- 通过`str`类型和自定义的占位符`${}$`生成的提示词模板，填充后结果为：

  ```python
  [
      UserMessage(role='user', content='你是一个精通数学领域的小助手，用户问题：1+1=?', name=None)
  ]
  ```
