## openjiuwen.core.security

`openjiuwen.core.security` 模块用于承载与安全相关的能力和扩展点（例如鉴权、审计、数据安全等）。

当前版本中，该包仅作为占位与扩展入口：

- 源码入口：`openjiuwen.core.security.__init__`，`__all__` 为空列表；
- 尚未对外导出具体类或函数。

> **说明**：
>
> - 之所以在 API 文档中保留本模块，是为了保证文档目录与源码目录一一对应，便于未来在该包中新增安全相关能力时快速补充文档；
> - 目前业务开发者无需直接使用本模块，对安全能力的实际使用请参考上层产品或接入层的文档。

当后续在 `openjiuwen.core.security` 中新增对外可用的安全组件时，本 README 会同步补充相应的类/方法说明。
