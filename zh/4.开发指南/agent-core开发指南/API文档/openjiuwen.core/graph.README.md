# graph

`openjiuwen.core.graph`是openJiuwen的工作流执行图。

**Classes**：

| CLASS              | DESCRIPTION            |
|--------------------|------------------------|
| [Graph](./graph/graph.md) | Graph 类是用于构建工作流执行图的基类。 |

**Alias**：

| ALIAS                     | DESCRIPTION                                                          |
|---------------------------|----------------------------------------------------------------------|
| [Router](./graph/graph.md) | Router为图或工作流的条件边的路由，用来决定条件边的目标节点或者目标节点列表。|
