# memory

`openjiuwen.core.memory`是openJiuwen框架的记忆引擎，提供Agent运行过程中记忆存取、处理等能力。

**Classes**：

| CLASS                                 | DESCRIPTION |
|---------------------------------------|-------------|
| [SysMemConfig](./memory/config.md)    | 记忆引擎系统级配置类。 |
| [MemoryConfig](./memory/config.md)    | 记忆引擎群组级配置类。 |
| [MemoryEngine](./memory/engine.md)    | 记忆引擎类。      |
| [BaseKVStore](./memory/store.md)      | kv数据库抽象基类。  |
| [BaseSemanticStore](./memory/store.md) | 语义召回抽象基类。   |
| [BaseDbStore](./memory/store.md)      | sql数据库抽象基类。 |
