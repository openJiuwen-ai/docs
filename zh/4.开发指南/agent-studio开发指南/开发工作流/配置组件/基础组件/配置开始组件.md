# 开始组件

开始组件是所有工作流的第一个节点（必要），是工作流的起点。
开始组件是工作流的入口，专门用于配置启动参数。开始组件自带一个表示用户输入的 `query` 参数，并支持用户按需添加或修改其他参数。

![image](../../../images/653c5c02-01a5-40d8-a266-3d4759ab1bfd.png)

# 配置组件

## 操作步骤

1. 进入openJiuwen平台主页。
2. 进入平台左侧导航栏的**工作流编排**模块。
3. 单击画布中“开始”组件，打开组件编辑界面。

![image](../../../images/4a09a8c5-21ff-47e2-bdd9-f5fc1feda6fc.png)

4. ​**添加或删除输入参数**​ ：单击`+`添加参数，单击`-`删除参数。

![image](../../../images/e42c7ada-6290-472a-a012-e63460438edc.png)

开始组件参数说明如下：

| 参数| 说明 |
| --- | --- |
| 左侧输入框 | 输出参数的key |
| 右侧输入框 | 数据类型：支持配置`String`、`Number`、`Object`  等多种类型的输入参数。|
| 是否为必选参数 | 设置参数是否必选。|
| 参数描述 | 参数的描述信息，帮助模型理解传入的参数含义。|
| 参数默认值 | 默认设置参数输入值，支持修改。|
