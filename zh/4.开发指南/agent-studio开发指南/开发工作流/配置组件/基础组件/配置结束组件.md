# 结束组件

结束节点用于输出工作流的结果。是工作流的最后一个节点（必要），是工作流的结束。在工作流运行结束后，输出将直接返回指定输出内容。
![image](../../../images/5cde4461-85ee-409e-baab-74ace0f9f21e.png)

# 配置组件

## 操作步骤

1. 进入openJiuwen平台主页。
2. 进入平台左侧导航栏的**工作流编排**模块。
3. 单击画布中“结束”组件，打开组件编辑界面。

![image](../../../images/e57a423f-d8a7-4fe1-8358-ba0919444561.png)

4. **添加或删除输入参数**​：单击`+`添加参数，单击 ![image](../../../images/9bc87254-e58f-48f6-9cf9-d7a946887e83.png)删除参数。

![image](../../../images/a8d5beeb-393d-4e5c-9fc1-a1573ea07047.png)

结束组件参数说明如下：

| 参数| 说明 |
| --- | --- |
| 左侧输入框 | 输出参数的key |
| 右侧输入框 | 数据类型：支持配置`String`、`Number`、`Object`  等多种类型的参数，变量值可以是固定值，也可以引用上游组件的输出结果。 |
